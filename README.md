# MotionBuilder-Script

Autodesk MotionBuilder Python scripts and utilities.

## Structure

- `Retargeter/` — Retargeting-related scripts and tools.

## Requirements

- Autodesk MotionBuilder (with Python scripting enabled)
- Python (version provided by MotionBuilder)

## Usage

Open MotionBuilder, then run scripts via the Python Editor or Tools menu, depending on the script.
